# Whisper Local 逐字稿

Windows WPF desktop app for creating transcripts from local video or audio files with a local Whisper model.

Project source:

```text
WhisperApp/
```

Run from the project folder:

```powershell
cd WhisperApp
dotnet run
```

Published executable:

```text
WhisperApp/bin/Release/net8.0-windows/publish/WhisperApp.exe
```

Install local Whisper dependencies:

```powershell
cd WhisperApp
.\scripts\install-whisper.ps1
```
